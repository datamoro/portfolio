National Franchise Incentive Program

USD 2.8M Managed

The Challenge

Managing the calculation and auditing of annual bonuses for 400+ franchises was a manual, error-prone process that lacked transparency. The finance team spent weeks consolidating Excel sheets, leading to delays and disputes.

The Solution

Engineered an end-to-end data solution automating complex auditing with SQL and Python. Built personalized real-time dashboards for franchisees to track their performance against targets.

Key Achievements

  • Automated 100% of the bonus calculation process, saving 3 weeks of manual work per cycle.
  • Reduced calculation errors to 0%, ensuring fair and transparent payouts.
  • Provided 400+ franchisees with daily updated performance metrics via Power BI Row-Level Security (RLS).
